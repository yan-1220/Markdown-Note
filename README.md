# 📝 Markdown Note Cloud

一個純前端的 Markdown 筆記應用，使用瀏覽器本地儲存，支援深色模式與即時預覽。

## ✨ 功能特色

- 📄 **Markdown 渲染** - 支援完整 Markdown 語法（包含表格）
- ✏️ **雙模式切換** - Markdown 閱讀模式 ⇄ 即時編輯模式
- 🔍 **全文檢索** - 即時搜尋標題和內容，關鍵字高亮顯示
- 📅 **時間標記** - 自動記錄建立時間與最後更新時間，列表顯示相對時間
- 👁️ **即時預覽** - 編輯時即時查看渲染結果
- 💾 **自動儲存** - 編輯後 1.5 秒自動儲存到本地
- 🌓 **完美深色模式** - 護眼深色界面設計
- 📱 **響應式設計** - 支援桌面和行動裝置
- 🔐 **安全保護** - 使用 DOMPurify 防止 XSS 攻擊
- 💻 **離線優先** - 無需網路連線即可使用

## 🚀 快速開始

### 立即使用（無需配置）

應用程式**開箱即用**，使用瀏覽器本地儲存（localStorage）保存筆記資料：

**方式 1: 直接開啟檔案**
```bash
# 在檔案總管中雙擊 index.html
# 或在命令列中執行：
start index.html  # Windows
open index.html   # macOS
xdg-open index.html  # Linux
```

**方式 2: 使用本地伺服器（推薦）**
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx http-server -p 8000
```

然後在瀏覽器開啟 `http://localhost:8000`

就這麼簡單！無需任何配置，立即開始使用。

## 📖 使用說明

### 建立筆記
1. 點擊左側面板的 **「➕ 新增筆記」** 按鈕
2. 新筆記會自動建立並顯示在列表中

### 搜尋筆記
1. 在左側面板的搜尋框輸入關鍵字
2. 即時過濾符合的筆記（搜尋標題和內容）
3. 符合的關鍵字會以黃色標記高亮顯示
4. 顯示找到的筆記數量
5. 按 `Esc` 鍵或點擊 `✕` 清除搜尋

### 時間標記
1. 筆記列表自動顯示最後更新時間（相對時間格式）
   - 剛剛、5 分鐘前、2 小時前、3 天前等
   - 超過 7 天顯示完整日期時間
2. 查看筆記時，標題下方顯示詳細時間資訊
   - **建立時間**：筆記首次建立的日期時間
   - **最後更新**：筆記最近修改的日期時間
3. 時間格式：YYYY-MM-DD HH:MM（例如：2025-11-09 14:30）

### 查看筆記
1. 點擊左側列表中的任一筆記
2. 自動進入 **Markdown 模式**，全螢幕顯示渲染結果

### 編輯筆記
1. 在 Markdown 模式下，點擊 **「✏️ 編輯」** 按鈕
2. 進入編輯模式：左側編輯器 + 右側即時預覽
3. 修改標題和內容，會在 1.5 秒後自動儲存
4. 點擊 **「✅ 完成」** 按鈕返回 Markdown 模式

### 兩種模式
- **Markdown 模式**（預設）：全螢幕顯示渲染結果，專注閱讀
- **編輯模式**：雙欄布局，左側編輯 + 右側即時預覽

### 刪除筆記
1. 選擇要刪除的筆記
2. 點擊右下角的 **「🗑️ 刪除筆記」** 按鈕
3. 確認刪除

### Markdown 語法範例

```markdown
# 一級標題
## 二級標題
### 三級標題

**粗體文字**
*斜體文字*

- 項目 1
- 項目 2
- 項目 3

1. 編號項目 1
2. 編號項目 2

[連結文字](https://example.com)

> 引用文字

\`inline code\`

\`\`\`javascript
// 程式碼區塊
const hello = "world";
\`\`\`
```

## 🏗️ 技術架構

- **HTML/CSS/JavaScript** - 純前端實作（單一檔案）
- **Tailwind CSS** - 快速美觀的 UI 設計
- **localStorage** - 瀏覽器本地儲存
- **marked.js** - Markdown 轉 HTML
- **DOMPurify** - XSS 防護（防止跨站腳本攻擊）

## 💾 資料儲存

- **本地儲存**：所有筆記儲存在瀏覽器的 localStorage 中
- **自動儲存**：編輯後 1.5 秒自動儲存
- **持久化**：資料永久保存在本機，除非清除瀏覽器資料
- **隱私保護**：所有資料僅保存在您的電腦上，不會上傳到任何伺服器

## 🔒 安全性

- **XSS 防護**：使用 DOMPurify 清理所有 Markdown 渲染的 HTML 輸出
- **本地優先**：所有資料僅儲存在本機，無需註冊或登入
- **無追蹤**：不收集任何使用者資料或分析資訊
